    Syntactic frame class URIs from the LexInfo ontology.

    Use as the value of @type on a synsem:SyntacticFrame node. LexInfo defines 80+ frame classes; this enum covers the high- and medium-priority ones identified for Etrog.

    Type Declaration

    • ReadonlyAdjectiveAttributiveFrame: "http://www.lexinfo.net/ontology/3.0/lexinfo#AdjectiveAttributiveFrame"

    • ReadonlyDitransitiveFrame: "http://www.lexinfo.net/ontology/3.0/lexinfo#DitransitiveFrame"

    • ReadonlyIntransitivePPFrame: "http://www.lexinfo.net/ontology/3.0/lexinfo#IntransitivePPFrame"

    • ReadonlyTransitiveFrame: "http://www.lexinfo.net/ontology/3.0/lexinfo#TransitiveFrame"

    • ReadonlyTransitivePPFrame: "http://www.lexinfo.net/ontology/3.0/lexinfo#TransitivePPFrame"

      Transitive verb + prepositional phrase (e.g. "she added salt to the stew").
