    Class LexicalEntryBuilder

    Fluent builder for ontolex:LexicalEntry objects.

    Example

    const entry = new LexicalEntryBuilder("urn:uuid:..." as URI, "en" as LanguageTag)
      .setCanonicalForm(form)
      .setPoS(LexInfoPoS.noun)
      .build();
