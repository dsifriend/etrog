    Class LexicalLinksetBuilder

    Fluent builder for lime:LexicalLinkset objects.

    Example

      .setReferenceDataset("https://example.org/ontology" as URI)
      .setConceptualDataset("https://example.org/concepts" as URI)
      .build();
