    Class LexicographicResourceBuilder

    Fluent builder for lexicog:LexicographicResource objects.

    Example

      .addTitle("Oxford English Dictionary", "en" as LanguageTag)
      .setLanguage("en" as LanguageTag)
      .addEntry(entry)
      .build();
